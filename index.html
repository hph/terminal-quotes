Terminal Quotes

This program was written as a replacement for Linux Mint's "Fortune Cookies". It prints a quote every time the terminal is opened. More quotes can easily be added.

Setup

Manual install (Linux)

Execute the following commands (just copy and paste):

git clone git://github.com/haukurpallh/terminal-quotes.git
mv terminal-quotes ~/.termquotes
echo python /home/$USER/.termquotes/termquotes.py >> ~/.bashrc

Adding more quotes

The procedure to add quotes is fairly simple. For example, open general.txt and enter the following lines:

[William Shakespeare]
To be or not to be: that is the question.

There can be no empty lines between the line containing the name of the author and the quotes and there has to be an empty line between an author's last quote and the next author's header ([William Shakespeare] in this case). Also, the last line of the file has to be empty. Different quotes from the same author are separated with & as in the following example:

[William Shakespeare]
Even you, Brutus?
&
To be or not to be: that is the question.

Usage

Every time you open a new terminal the program will print a quote. If you also want to be able to print a quote whenever you want, execute the following command:

echo alias q="python /home/$USER/.termquotes/termquotes.py >> ~/.baschrc"

Now enter the q command to get a random quote.
